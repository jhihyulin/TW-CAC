# TW-University-result-query

## 關於

目前為112年資料，資料由公告網站爬取，如有錯誤請提出Issue，謝謝！

可使用應試號碼查詢
- 大學繁星推薦招生錄取結果
- 大學申請入學第一階段篩選結果
- 科技校院日間部四年制申請入學聯合招生第一階段篩選結果
    - 若有通過任何一所科技校院日間部四年制申請入學聯合招生第一階段篩選，則會顯示去識別化的姓名，如「林Ｏ宇」。
